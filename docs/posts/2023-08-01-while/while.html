The Story

Once upon a time in ancient India, a wise courtier presented a challenge to his king. He placed a single grain of rice on the first square of a chessboard and then asked the king to double the number of grains on each subsequent square until all 64 squares were filled. In the beginning, the king thought that it was a meaningless amount of rice, but then he realized the astronomical sum this would amount to.

This story illustrates the power of exponential growth. But in this case, I just want to focus on this tale, one of the most common stories related to the origin of chess. To make this more entertaining, let’s compute the actual number using Python.

Mathematical Explanation

The number of grains on the chessboard follows exponential growth, where each term is double the previous one. The total number of grains can be computed using the formula for the sum of a geometric progression, but don’t worry, we don’t need to use this formula. We now have computers that can actually compute the number of grains by adding one by one! Crazy, isn’t it?

Python Code to Compute the Total Number of Grains

We can compute the total number of grains using a while loop in Python:

# Initialize variables
square_number = 1  # The number of the current square
grains_on_square = 1  # The number of grains on the current square
total_grains = 0  # The total number of grains so far

# Loop through each square on the chessboard
while square_number <= 64:
    # Add the number of grains on the current square to the total
    total_grains += grains_on_square
    # Double the number of grains on the current square for the next square
    grains_on_square *= 2
    # Move to the next square
    square_number += 1

print(f"The total number of grains on the chessboard is: \n {total_grains:,}")

# Convert to kilograms
total_weight_kilograms = total_weight_grams / 1000

# Convert to metric tons
total_weight_metric_tons = total_weight_kilograms / 1000

Conclusion

The chessboard problem illustrates how quickly numbers can grow when they double with each step.

This is a good example of how to use Python to solve a problem that would be difficult to solve by hand.
